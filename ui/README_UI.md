# Mini Library UI (Optional Bonus)

## Tech
Vanilla HTML/CSS/JS (no frameworks).

## Purpose
Simple browser UI for managing books without a backend.

## Run
Just open `ui/index.html` in a browser.

## Persistence
Books are stored in the browser using `localStorage`.

## Notes
- UI data does not sync with the CLI `books.json`
- No backend or server is required

## Features
- Add book (title, author, year)
- Search by title/author
- Delete by ID (button)
